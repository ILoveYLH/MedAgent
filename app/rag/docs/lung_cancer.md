# 肺癌 CT 影像特征与诊断指南

## 概述

肺癌是全球发病率和死亡率最高的恶性肿瘤之一，CT 检查是肺癌诊断、分期和随访的核心影像学手段。早期发现和准确诊断对改善预后至关重要。

## CT 影像特征

### 早期肺癌（IA期）

- **纯磨玻璃结节（pGGN）**：病灶密度低于肺实质，不遮盖血管和支气管，常见于原位腺癌和微浸润腺癌。
- **部分实性结节（mGGN）**：同时含有磨玻璃成分和实性成分，实性比例 ≥25% 时恶性风险显著升高。
- **实性结节**：直径 ≤1cm 的小实性结节，边缘分叶或毛刺时需高度警惕。

### 进展期肺癌特征

- **肿块形态**：分叶征、毛刺征（短细毛刺提示浸润性生长）
- **边缘特征**：边界不规则，胸膜凹陷征（V 字形或鱼钩形）
- **内部结构**：空气支气管征、空泡征、坏死腔形成
- **密度特征**：密度不均匀，可出现钙化（营养不良性钙化）
- **血管征**：血管集束征，肿瘤周围血管向病灶汇聚

### 侵犯和转移征象

- **纵隔淋巴结转移**：短径 ≥1cm 或融合淋巴结
- **胸膜侵犯**：胸腔积液、胸膜增厚结节
- **对侧肺转移**：多发随机分布结节
- **骨转移**：溶骨性或成骨性破坏

## 病理分型与CT对应

| 病理类型 | CT特征 |
|---------|--------|
| 腺癌 | 磨玻璃为主，边界不清，胸膜凹陷 |
| 鳞癌 | 中央型，肿块较大，空洞常见 |
| 小细胞癌 | 中央型，纵隔淋巴结肿大为主 |
| 大细胞癌 | 周围型，体积较大，坏死常见 |

## TNM 分期要点

- **T分期**：根据肿瘤大小（T1: ≤3cm，T2: 3-5cm，T3: 5-7cm，T4: >7cm 或侵犯重要结构）
- **N分期**：无淋巴结转移（N0），同侧支气管周围/肺门（N1），同侧纵隔/隆突下（N2），对侧纵隔/锁骨上（N3）
- **M分期**：无远处转移（M0），胸腔内播散（M1a），单发胸外转移（M1b），多发胸外转移（M1c）

## 诊断标准与 Lung-RADS

根据 ACR Lung-RADS（肺部影像报告和数据系统）：

- **Lung-RADS 1**：无结节或确定良性
- **Lung-RADS 2**：良性可能性大，建议12个月随访
- **Lung-RADS 3**：可能良性，建议6个月随访
- **Lung-RADS 4A**：可疑恶性，建议3个月随访或PET-CT
- **Lung-RADS 4B**：高度可疑恶性，建议组织活检
- **Lung-RADS 4X**：伴有其他恶性特征，建议活检

## 鉴别诊断

肺癌需与以下疾病鉴别：
1. **肺结核球**：边界清楚，可见卫星灶，钙化多见，增强无明显强化
2. **肺炎性假瘤**：边界较清，密度均匀，无明显分叶和毛刺
3. **错构瘤**：爆米花样钙化，脂肪成分（CT值 <-50HU）
4. **转移瘤**：多发，边界清楚，有原发肿瘤史

## 随访建议

- 实性结节 <6mm：无需常规随访（低危人群）
- 实性结节 6-8mm：6-12个月CT随访
- 实性结节 >8mm：3个月CT/PET-CT/活检
- 磨玻璃结节 >6mm：6个月CT，之后每年CT随访至少5年
