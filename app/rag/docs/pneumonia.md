# 肺炎 CT 影像特征与诊断指南

## 概述

肺炎是由细菌、病毒、真菌等病原体或理化因素引起的肺实质炎症，CT 是评估肺炎范围、严重程度和并发症的重要工具，尤其在普通 X 线检查阴性时具有重要价值。

## 主要CT影像征象

### 气腔型肺炎（大叶性/节段性）

- **实变影**：密度均匀增高，可见**空气支气管征**（实变背景中可见含气支气管影），边界沿叶间裂清楚
- **磨玻璃影**：密度轻度增高，可见血管和支气管影，提示炎性渗出或充血
- **分布特点**：按肺叶或肺段分布，多见于下叶，邻近胸膜面

### 支气管肺炎（小叶性肺炎）

- **多发斑片影**：双肺散在分布，中下叶为主，多沿支气管血管束分布
- **小叶中心结节**："树芽征"（支气管壁增厚伴分泌物）
- **支气管壁增厚**：细支气管炎表现，管腔可见黏液嵌塞
- **磨玻璃影**：弥漫分布，提示间质性肺炎可能

### 间质性肺炎

- **磨玻璃影**：双肺弥漫或胸膜下分布
- **网格影**：小叶间隔增厚，形成网状改变
- **蜂窝影**：晚期纤维化，多见于基底部和胸膜下
- **牵张性支气管扩张**：纤维化牵拉所致

## 按病原体分类的CT特征

### 细菌性肺炎

- **肺炎链球菌**：大叶性实变，空气支气管征，单叶分布，治疗后2-4周吸收
- **金黄色葡萄球菌**：多发空洞，脓胸，气胸，脓肿形成
- **克雷伯菌**：上叶好发，"膨胀性实变"，坏死空洞常见，重症

### 病毒性肺炎

- **COVID-19**：双肺外周、胸膜下磨玻璃影，铺路石征，进展期弥漫实变
- **流感病毒**：双肺弥漫磨玻璃影，支气管壁增厚
- **腺病毒**：多叶段实变，可出现坏死腔

### 非典型肺炎

- **支原体肺炎**：单侧或双侧树芽征，网格影，轻度磨玻璃影，以中下肺野为主
- **衣原体肺炎**：表现类似支原体，症状与影像不匹配（症状重，影像轻）

### 真菌性肺炎（免疫低下患者）

- **曲霉菌**：实变影周围磨玻璃晕（晕征），空洞内新月征（新月征）
- **隐球菌**：结节或肿块，边界不清，可见空洞
- **肺孢子菌**：双肺弥漫磨玻璃影，从肺门向周围扩散

## 严重程度评估

CT 严重程度评分（CT Severity Score, CT-SS）：
- 评估双肺各叶受累范围
- 0分：无异常；1-5分：轻度；6-10分：中度；11分以上：重度
- 受累肺叶 >50% 提示重症，需ICU治疗

## 并发症CT表现

- **胸腔积液**：游离积液或包裹性积液
- **脓胸**：胸膜增厚强化，积液密度高（CT值 >20HU）
- **肺脓肿**：厚壁空洞，内壁不规则，可见气液平面
- **支气管胸膜瘘**：肺底部气液平，胸腔内气体

## 鉴别诊断

| 疾病 | 鉴别要点 |
|------|---------|
| 肺癌 | 分叶毛刺，淋巴结肿大，进行性增大，无发热病史 |
| 肺结核 | 上叶好发，卫星灶，钙化，空洞薄壁 |
| 肺水肿 | 蝶翼状分布，心脏增大，治疗后迅速变化 |
| 肺梗死 | 胸膜基底楔形实变，深静脉血栓史 |

## 影像随访建议

- 轻症社区获得性肺炎：治疗后 4-6 周 X 线复查
- 重症或无改善者：1-2 周 CT 复查
- 45岁以上男性单侧肺炎：排除阻塞性肺炎，治疗后 4-6 周 CT 复查
- 复发性肺炎：需排除结构性肺病（支扩、异物）
